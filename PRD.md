# Tend

A behavioral intelligence platform designed for nightly reflection, planning, and adaptive habit coaching.

## Core Idea
Help users close the day intentionally in 5–10 minutes.

## Audience
Adults 50+, often high-performing but neurodivergent (ADHD / executive function friction).

## Nightly Flow
1. Arrival check (mood, energy, stress)
2. One win
3. One gratitude
4. Brain dump
5. Lesson learned
6. Tomorrow's top 1–3 priorities
7. If–then plan
8. Close signal

## Core Metrics
- Nightly completion rate
- D1 retention
- D7 retention
- Weekly summary views

## Platform Architecture
- GitHub (source of truth)
- Codex (development)
- Base44 (app surface)
- Supabase (backend)
- PostHog (analytics)
- HubSpot (CRM)
- Resend (email)
- n8n (automation)
